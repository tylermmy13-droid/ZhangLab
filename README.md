# Zhang Lab Website

This repository contains the Zhang Lab website based on the SpaceLab Astro template.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The site introduces the cardiovascular medicine, vascular regeneration, organoid-chip, and AI-enabled active health work of Zhang Lab at Xinhua Hospital affiliated to Shanghai Jiao Tong University School of Medicine.
