## English

This repository contains the Zhang Lab website based on the SpaceLab Astro template.

The site introduces the **cardiovascular medicine, vascular regeneration, organoid-chip, and AI-enabled active health** research at Zhang Lab, Xinhua Hospital affiliated to Shanghai Jiao Tong University School of Medicine.

### Local Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```
