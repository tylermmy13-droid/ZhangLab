## English

This is the official website of **Zhang Lab**, Xinhua Hospital affiliated to Shanghai Jiao Tong University School of Medicine.

The site introduces our research in **cardiovascular medicine, vascular regeneration, organoid-chip platforms, and AI-enabled active health**, as well as team members, publications, and news.

Built with [Astro](https://astro.build).

### Local Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```
